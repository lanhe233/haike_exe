四种格式的字体：
EOT -- 微软开发，IE专用
WOFF -- web字体中最佳格式，如今W3C组织推荐，ie9+支持
TTF -- 微软和苹果共同开发，两种操作系统中最常见的一种字体，ie9+支持
SVG -- 用于SVG字体渲染的一种格式，由W3C制定的开放标准的图形格式，chrome/safari支持

@font-face{
	font-family: "";
	src: url(xxx.eot); /* ie9兼容模式 */
	src: url(xxx.eot?) format("embedded-opentype"),
		 url(xxx.woff) format("woff"),				 
		 url(xxx.ttf) format("truetype"),
		 url(xxx.svg) format("svg");
	font-weight: normal;
	font-style: normal; 	/* 斜体等等 */
}
